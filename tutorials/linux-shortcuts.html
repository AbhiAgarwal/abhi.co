linux shortcuts

iptables

Accepting a port: 

[sudo] iptables -A INPUT -i eth0 -p tcp --sport 3000 -j ACCEPT

Setting up port routing using (from port 80 to 8000): 

[sudo] iptables -t nat -A PREROUTING -p tcp --dport 80 -j REDIRECT --to-ports 8000

Deleting existing rules: 

[sudo] iptables -F
